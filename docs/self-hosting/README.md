# Self-Hosting

You can run your own instance of Verbal Warning Logger on any machine that supports Python 3.10+. The bot stores all warning and poll data locally in SQLite files — the only external dependency is an optional Supabase connection for Auttaja historical data.

## Steps at a glance

1. [Prerequisites](prerequisites.md) — confirm Python version, install pip, etc.
2. [Discord Bot Setup](discord-setup.md) — create the application and get your token.
3. [Installation](installation.md) — clone the repo and install dependencies.
4. [Configuration](configuration.md) — create your `.env` file.
5. [Running the Bot](running.md) — start the bot and verify it's working.
