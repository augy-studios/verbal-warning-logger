# How to Use

Verbal Warning Logger uses Discord's slash command interface throughout. All commands appear in the `/` menu and include built-in descriptions for every option.

## Command groups

| Group | Purpose |
|-------|---------|
| [`/verbal`](verbal-warnings.md) | Add, view, edit, and delete verbal warnings |
| [`/auttaja`](auttaja.md) | Browse historical punishments from the Auttaja bot |
| [`/poll`](polls.md) | Create and manage staff evaluation polls |
| [`/poll_template`](poll-templates.md) | Save and reuse poll structures |
| [`/retrieveids`](utility.md) | Bulk-export channel, role, or user IDs |
| `/ping` | Check bot latency |
| `/about` | Display bot information |
| `/help` | Interactive in-Discord help menu |

## Who can run commands?

Most moderation commands require the configured staff role or above. See the [Permissions](permissions.md) page for a full breakdown.
